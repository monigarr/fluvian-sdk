# Contributing to Fluvian SDK

This repository is **proprietary** and shared primarily for **evaluation** under the terms in [LICENSE](LICENSE). Read [docs/EVALUATION_TERMS.md](docs/EVALUATION_TERMS.md) before using or sharing the code.

## How to participate

- **Bugs and docs:** Open a GitHub issue with a minimal repro or a clear description of the documentation gap.
- **Security:** Do **not** open public issues for vulnerabilities. Use the process in [SECURITY.md](SECURITY.md).
- **Code contributions:** Unsolicited pull requests may not be accepted. Email **monigarr@monigarr.com** first if you intend to contribute patches so licensing and IP boundaries can be agreed in writing.
