# Pricing – Fluvian SDK

Fluvian SDK follows an **open core** model. You can evaluate and prototype with the full architecture for free. Upgrade to PRO when you need production‑ready AI optimization, private artifacts, and support.

---

## Open Core – Free

✅ Full SDK architecture (Media3, DRM‑ready, QoS rule engine)  
✅ AI abstraction layer (facades for your own models)  
✅ Demo app + public test streams  
✅ Local CI scripts and documentation  

**Best for:** Evaluation, internal prototypes, learning the system design.

---

## PRO SDK – $49–$299 / month

Everything in Open Core, plus:

- AI‑powered QoS optimization (on‑device + cloud inference)  
- Bandwidth prediction and advanced playback tuning  
- Private SDK artifacts (AAR) delivered via protected repository  
- Priority updates and email support  
- Access to fleet **Learn** features (where offered)  

**Pricing tiers** depend on:
- Number of apps / seats
- Monthly active users or streams
- Specific AI features enabled

> Final price confirmed in writing after a brief scoping call.

**Best for:** Teams shipping production apps that need better streaming performance and AI assistance.

---

## Enterprise – Custom pricing

All PRO features, plus:

- White‑label SDK (your branding, our engine)  
- Custom player factories and streaming architecture  
- Performance optimization engagements  
- Dedicated support channel + SLA  
- NDAs and custom licensing terms  

**Best for:** OTT platforms, live sports broadcasters, large‑scale deployments.
